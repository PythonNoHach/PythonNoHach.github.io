Python学习计划
发表于 2019-07-19| 更新于 2019-07-19

学习Python的原因

  • 已经学习了JAVA,手头没有计算机只有安卓手机,使用termux学习Python比较方便

  • 想学会爬虫做各种各样的事情

  • 有一定Linux基础,学起来不会很吃力

学习Python的途径

  • Python官方文档和购置的学习书籍

  • 安卓端使用QPython3-IDE编辑和运行Python程序

  • 安卓端通过termux在终端运行Python

  • 通过哔哩哔哩视频教程学习Python

如何安排学习Python的时间

  • 6hours/d

  • 30d完成
